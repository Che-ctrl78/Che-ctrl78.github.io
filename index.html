Engine Cylinder

A simple model demonstrating how an engine cylinder functions. Requires two M3x15mm screws and an M3 hex key, bit, or Allen wrench.

Download
Instructions:
  • Print all parts using PLA at 0.2mm layer height.
  • Use support material for the piston if needed.
  • Arrange parts as seen in image.
  • Place the large peg inside the large hole to attach the piston to the moving axle.
  • Using a coin or a flathead screwdriver, screw the printed screw into the top hole on the rotating base.
  • Assemble body using a screw in the hole on the bottom.
  • Use the second screw on the top of the rotating cylindrical part.
  • Do not fully tighten screws.

Gear set

A small demonstration of gears and how they interact with each other. Requires two M3x15mm screws and an M3 hex key, bit, or Allen wrench.

Download
Instructions:
  • Print all parts using PLA at 0.2mm layer height.
  • The smaller gear goes in the middle with the larger gears next to it on the piece with pegs.
  • Place the other piece over the gears.
  • Assemble using one screw in each of the screwholes over the gears on the top, not on the side with pegs.
  • Do not fully tighten screws.

Tensegrity Table

A tensegrity table, which holds itself up using only strings. Requires three small strings which can be made of any material.

Download
Instructions:
  • Print all parts using PLA at 0.2mm layer height.
  • One piece sits vertically on the table while the other is inverted and rotated above it.
  • Attach one piece of string between the two central holes on each part.
  • Connect the holes on each side vertically using longer pieces of string.
