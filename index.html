Engine Cylinder

A simple model demonstrating how an engine cylinder functions. Requires two M3x15mm screws and an M3 hex key, bit, or Allen wrench.

Download
Instructions:
  • Print all parts disconnected using PLA at 0.2mm layer height. Do not change size or scale.
  • Models are designed to be printed without supports, so arrange on print bed in optimal positions
  • Arrange parts as seen in image.
  • Place the wristpin inside the large hole of the piston while the connecting rod is attached to make the unit one.
  • Using a flathead screwdriver, screw the crank shaft rivet into the body fastening it to the body.
  • Assemble crankshaft to body using a screw in the hole on the bottom.
  • Use the second screw on the top of the rotating cylindrical part.
  • The bottom screw should be snug and the top screw should have a little clearance to allow the top cap to act as a rotating handle

Gear set

A small demonstration of gears and how they interact with each other. Requires two M3x15mm screws and an M3 hex key, bit, or Allen wrench.

Download
Instructions:
  • Print all parts disconnected using PLA at 0.2mm layer height. Do not change size.
  • The smaller gear goes in the middle with the larger gears next to it on the piece with pegs.
  • Place the cover on top of the base with the gears in it.
  • Assemble by puttng m3 bolts in each of the screw holes. Make sure the bolt goes on the covering plate side instead of the side with the pegs.
  • Make sure the bolts are snug, but DO NOT overtighten

Tensegrity Table

A tensegrity table, which holds itself up using only strings. Requires three small strings which can be made of any material.

Download
Instructions:
  • Print all parts separately using PLA at 0.2mm layer height.
  • One piece sits vertically on the table while the other is inverted and rotated above it.
  • Attach one piece of string between the two central holes on each part.
  • Connect the holes on each side vertically using longer pieces of string.
  • After attaching the central string, you may need to use an object to prop one side of the table up to attach the supportng strings
